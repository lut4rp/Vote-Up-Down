Vote Up/Down

Google Summer of Code 2009 project for Drupal
Authored by Pratul Kalia, Mentored by Simon Roberts

Introduction

It is a Drupal 6 module for all your voting requirements!

Dependencies

Vote Up/Down needs the Voting API module, get it from http://drupal.org/project/votingapi

Download

You can download the current nightly build in either zip or tar formats.
Please note that these may/may not be in a working condition. The project is still being actively developed and should NOT BE used in a production environment.

Install

- Download and install Voting API
- Then, download and install Vote Up/Down
- Hack!

Bugs/Issues

All issues should be reported on the Drupal.org issue tracker
- http://drupal.org/project/issues/vote_up_down

Please mark the issues for version 6.x-2.x-dev
